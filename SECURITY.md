# Security Policy

## Reporting A Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report security problems directly to Salman:

- Email: [creativecoder.crco@gmail.com](mailto:creativecoder.crco@gmail.com)

If possible, include:

- a short summary of the issue
- affected area or files
- steps to reproduce
- impact
- any proof of concept or logs
- suggested mitigation, if you have one

## Response Expectations

ReacherX is an actively developed project, but response times may vary.

The goal is to:

- acknowledge legitimate reports reasonably quickly
- verify the issue
- decide on remediation steps
- ship a fix or mitigation when appropriate

## Disclosure

Please avoid public disclosure until the issue has been reviewed and a fix or mitigation plan is in place.

Coordinated disclosure is appreciated.
