## GET Keyword Ideas

**Endpoint**

```

[https://api.bishopi.io/keyword_ideas/?keywords=milke](https://api.bishopi.io/keyword_ideas/?keywords=milke) shake, domain tools

```

### Cost

**10 Credits per request**

### Description

Search terms that are relevant to the product or service categories of the specified keywords.

### Headers

| Key           | Value                |
| ------------- | -------------------- |
| Authorization | Api-Key `<your key>` |

### Params

| Name     | Value                     |
| -------- | ------------------------- |
| keywords | milke shake, domain tools |

```

curl --location 'https://api.bishopi.io/keyword_ideas/?keywords=milke%20shake%2C%20domain%20tools' \
--header 'Authorization: Api-Key <your key>'
```
